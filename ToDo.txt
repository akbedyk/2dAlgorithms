
# Add 2d Algorithms:


1. Определение точки внутри / снаружи области (фигуры). Разбиение фигуры на выпуклые части:
   - Алгоритм Грэхема O(n*log(n))
   - Алгоритма Джарвиса O(H*n) H = 1...n


   
2. Обход фигуры по часовой / против часовой стрелки
3. Поиск пути Дейкстра, A+, Алгоритм прыжковой точки
4. Поиск пути на больших картах
   - разбиение на ячейки, выделение граней, через которые проходят большие фигуры
   - отсечение граней, где нет переходов на ячейку
   - отсечение граней связей, где нет прохода сквозь ячейку
   - кэш основных путей и их стоимости
   - кэш найденных путей между объектами
